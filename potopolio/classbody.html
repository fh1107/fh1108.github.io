HTML

HTMl 이란 Hyper Text + Markup Language로써 하이퍼 링크등이 가능하도록 마크업 이용 웹문서의 구조로 표현된 것이다. 여기서 Hyper Text란 참조를 통해 한 문서에서 다른 문서로 즉시 접근할 수 있는 텍스트 이다. 그리고 Markup Language는 markup(문서에 부가적인 정볼르 표시하기 위한 표기방식) +Language 로써 태그 등을 이용하여 문서나 데이터의 구조를 명기하는 언어이다.

HTML의 문서 구성은 엘리먼트들의 모임이다. 여기서 엘리먼트란 <요소이름> + 내용(텍스트) + <요소이름> 혹은 시작태그 + 내용 + 종료태그 라고 할수 있다. HTML 문서의 기본단위로써 하나의 엘리먼트 안에 다수 엘리먼트가 중복이 가능하고 HTML문서의 논리적인 구성요소 이다.

ex) <h1>애국가</h1>

요소안에 내용이 없는 경우도 있다. ex) <br>

요소안에 요소가 들어가는 것들도 있다. <요소이름> <요소이름>내용(콘텐츠)

요소 이름은 속성을 가지고 있다. 속성 = 객체/제품의 특성/성질

MVC : 화면(view),데이터(model),기능(control)로 구분하여 인터넷 프로그램을 개발한다. 여기서 화면은 CSS를 말하며 데이터는 HTML 기능은 java script를 말한다.

HTML의 기본구조는 DTD(Document Type Dfinition)으로 되어있다. HTML의 문서의 최상위 요소는 <<html>> 이고 다음 <head> 그리고 <body> 로 구성되어 있다.
<html>은 웹문서의 시작과 끝을 설정하며 차상위 요소로 <head> , <body> 요소를 하나씩 포함한다.
<head>은 HTML 문서를 설명하는 요소로 하위 요소는 <meta> , <title> , <link> , <style>, <script>가 있다.
<body>는 HTML 문서의 실제 내용이 작성되는 요소로써 다양한 태그를 이용하여 엘리먼트를 작성한다.

<head>는 html문서에 대한 정볼르 메타 데이터의 형식으로 기록하는 것이다.
<head>의 종류는 <title> , <link> , <style> ,<script>, <meta>가 있다.
<title>은 문서의 표제를 설정하는 필수 요소로 브라우저 타이틀바에 표시
<style> 은 html 문서 엘리먼트의 배경색 , 글자 색깔등 스타일을 설정할때 사용 된다.ex)CSS
<link>는 css스타일을 파일로 관리할때 html에서 css파일 연결시 사용
<script>은 클라이언트의 동작 처리를 위한 자바스크립트를 기술할때 사용한다.
<meta>에서는 속성이 name ,contnent,charset,http-equiv가 있다.
name은 문서 설명과 검색 키워드 관련 메타정보 , 뷰포트지정설정을 한다.
http-equiv는 http서버가 http 응답메세지 헤더를 위한 정보수집시 사용
content는 name,http-equiv속성과 연관된 text데이터
charset은 html문서의 인코딩관련 메타정보 설정

<body>는 html문서의 본문을 수록하는 요소이다. 뷰로트에 <body>내용이 표현되고 반드시 한번 사용한다.
<body>속성에 이벤트를 발생시키는 것이 있는데 사용자 정의는 자바스크립트로 정의한다.
onblur 현재창 닫기,다른 페이지 이동링크 클릭,document.write()실행시
onerror 런타임스크립트 오류,보안기준위반등 페이지 오류시 실행
onfocus 해당 브라우저가 포커스 받았을때 실행
onload 문서 DOM,이미지 등의 html 로딩이 완료되었을 때 발생
onunload 사용자가 페이지를 떠날때 발생

<h>는 소제목 헤딩을 표현하는 태그 hn n= 1,2,...6 글자가 작을수록 중요하다 판단 h1>h2>h3>h4>h5>h6 순 글자크기
<p>는 문단을 구분하기 위한 태그이고 줄바꿈과 새로운 서식이 시작된다.
<br/> 하나의 줄바꿈을 위한 태그이고 종료가 없는 empty tag이다.
<hr/> 가로줄을 추가할 때 사용한고 이것도 마찬가지로 종료가 없는 empty tag이다.
<b>는 해당 콘텐츠를 강조글꼴로 표기한다.
<strong>는 해당 콘텐츠를강조글꼴로 표기(b와 유사)
<em> 해당 콘텐츠를 강조해서이탤릭(기울림) 글꼴로 표기
<code>는 컴퓨터 코드를 표기한다.
<samp>는 컴퓨터 프로그램의 sample 결과물을 정의한다.
<i>는 해당 콘텐츠를 이탤릭글꼴로 표기한다.
<ins>는 해당 콘텐츠를 밑줄 글꼴로 표기한다.
<del>는 해당 콘텐츠를 취소 글꼴로 표기한다.
<sup>는 해당 콘텐츠를 위첨자로 표기한다.
<sub>는 해당 콘텐츠를 아래첨자로 표기한다.
<small>는 해당 콘텐츠를 작은 글꼴로 표기한다.
<mark>는 해당 콘텐츠를 시각적으로 강조글꼴로 표기한다.
<time>는 날짜와 시간정보를 표현하는 태그3:03이다.
<address>는 주소(이메일)정보를 표현하는 태그이다.

<li>는 세부 항목의 나열을 통한 구조화를 하기위한 태그이다. 이것은 순서화된 목록인 <ol> 과 비순서화 된 <ul> 나뉜다.
<ol>에서 속성은 type , start, reversed가 있는데 type는 숫자 혹은 대문자 ,소문자 대문로마자,소문소마자로 표현할것 인지 설정하는것 이다.
start는 시작를 어느것부터 할건지 설정하는 것이다. 그리고 reversed는 내림차순으로 설정하기 위한 속성이다.
<ul>에서는 style,value 속성이 있는데 style은 리스트 아이템 마커를 무엇을 할것인지 설정하기 위한 속성이다.
value는 시작번호를 설정하기 위한 속성이다.

<dl> 기술할 리스트를 정의한다.
<dt> 기술할 리스트의 용어를 정의한다.
<dd> 기술할 리스트의 용어를 설명한다.

표를 작성하기 위한 요소에는 <table> , <tr>,<th>,<td>가 필요하다.
<table>은 테이블의 시작과 끝을 정의한다.
<tr>은 테이블에서 사용되는 하나의 행을 정의한다.
<th>은 테이블의 머리말 셀을 정의 한다. <th>에서는 속성이 addr,colspan,rowspan이 있다.
addr는 머리글 셀의 내용을 축약 버전으로 저장시키는 것이다.
colspan은 헤더셀이 확장되어야 하는 열의 수를 지정하는 것이다.
rowspan은 헤더셀이 확장되어야 하는 행의 수를 지정하는 것이다.
<td>는 테이블의 데이터 셀을 정의 한다. 속성은 colspan,rowspan이 있는데 <th>의 속성의 내용과 같다.
HTML5에서 세멘틱 콘셉트라고 추가된 <caption> , <thead> , <tbody>,<tfoot> 가 있다. <caption>은 표제목,<thead>는 표머리,<tbody>는 표 몸통,<tfoot>는 표발을 뜻한다.

<div>와<span>은 홈페이지의 공간을 일정 부분으로 분할하여 공간 배치하는 태그이다.
<div>는 블록 형식으로 공간을 분할 하고 각 영역은 새로운 행에 작성되며 줄바꿈 태그 없이 스스로 줄바꿈을 시행한다. 요소 영역은 세로로 진행되며 하위 요소로 블록 요소 및 인라인 요소 포함이 가능하다. 문단과 문단을 구분하기 위한 요소이다. <p>와의 차이점은 태그 앞뒤로 빈줄이 없다.
<span>은 인라인 형식의 공간 분할을 하고 각 영역은 줄바꿈 요소가 없는 경우 가로로 계속 진행되며 각 영역은 블록 요소 안에 단독 사용되지 않는 경우 가로로 계속 진행되고 블록 형식 요소의 하부 요소로 포함되어 사용이 가능하고 하위 요소로 불록 요슨느 불가능 하다.(단, <a> 인라인 하부요소로 <h1> ~ <h6>,<p> 등 불록 요소 가능하다.)
<div>요소와 유사 하지만 인라인 요소라는 점과 문장 일부만 선택 혹은 태그안의 내용만큼만 너비를 차지한다는 것이다.

시멘틱이란 문서에 정보의 의미를 부여한느 것이다. HTML5에서 <div>요소와 유사한 요소를 개발하여 부여한것으로 로봇 검색이 용이하도록 설정하였다.
기본구조는 <header> , <nav> , <main> ,<section> , <article> , <aside> ,<nav> , <footer> 가 있으며 순서 변경도 가능하다.
<body> : 몸통 기술하는 시멘틱 요소이다.
<main> : 메인 컨텐츠이며 한페이당 한번만 사용 가능 하고 MS-Explorer12이상만 가능하고 지금은 <section>으로 대체 사용한다.
<header> : 머리말 영역으로 웹 페이지 상단의 로고, 메뉴 등 머리말 영역을 지정한다.
<nav> : 메뉴 영역으로 웹페이지 내부 및 외부 이동을 위한 내비게이션 메뉴영역을 지정하며 검색 로봇 등에게 문서 이동관련 정보를 제공한다.
<section> : 내용 구분 영역으로써 웹 페이지의 구획을 지정 하나의 제목을 중심으로 내용들을 그룹화 하고 <section>안에 다수 중첩 가능하다.
<article> : 독립 컨텐츠로써 페이지내 독립적인 콘텐츠에 해당하는 영역을 지정한다.
<aside> : 보조영역으로 주로 페이지의 좌,우 측면에 배너광고,즐겨찾기 메뉴등 보조영역을 지정하고 section,article태그안의 내용을 간접적으로 관련된 내용 지정
<footer> : 꼬리말 영역으로 페이지 하단의 저작권, 작성자 주소,연락처 등 꼬리말을 지정한다.

<a> 문서간 문서내 등 특정 위치로 이동해주는 요소이다. 속성은 href, download,target이 있다.
href는 URL값을 넣어서 링크가 연결될 페이지의 URL을 지정해주는 것이다.
download는 링크를 클릭할 때 해당 파일을 다운로드 하게 해주는 것이다.
target은 연결 문서를 보여줄 장소를 지정하는 것이다
blank는 새로운 윈도우/탭을 여는것이다.
parent는 부모프레임을 여는것이다.
self는 현재 윈도우와 탭에서 여는것이다.
top은 body전체에서 여는것이다.
앵커에도 스타일이 있는데 종류를 보면 a:link , a:visited , a:hover , a:active가 있다.
a:link는 사용중인 브라우저가 해당링크를 방문하기 전 상태를 지정하기 위한 스타일이다.
a:visited 는 사용중인 브라우저가 해당 링크를 방문한 이후의 상태를 지정하기 위한 스타일이다.
a:hover는 마우스로 해당링크 위에 갖다 놓고 있는 경우에 어떻게 변하는지를 지정하는 스타일이다.
a:active는 마우스로 해당링크를 클릭하고 있는 순간에 어떻게 변하는 지를 지정하는 스타일이다.

<img>는 이미지 미디어를 출력할때 사용하는 태그이다. 속성으로는 src,alt,width,height,ismap,figure,figcaption이 있다.
src는 이미지의 URL을 지정하는 것이다.
alt는 해당 이미지 로딩 실패시 대안적 설명을 지정하는 것이다
width는 이미지 너비를 지정하는것이다(css사용 권장)
height는 이미지 높이를 지정하는 것이다(css사용 권장)
ismap은 서버 사이드 이미지맵을 지정하는 것이다.
figure은 이미지와 설명 텍스트를 하나의 그룹으로 정의하는 것이다.
figcaption은 이미지 제목을 표시하는 것이다.

이미지맵을 정의하는 것은 하나의 이미지에 다수 영역을 설정하여 영역별 링크를 연결하는 것이다.
<map>은 클라이언트 측면의 이미지 맵을 저의 이미지맵은 클릭할수 있는 영역을 다수 보유 하는것 이다.
속성 값으로는 name 즉 이미지 맵의 이름을 정하는 것이다.
<area>는 이미지 맵안에서 클릭할 수 있는 영역을 정의 하는 것이다.
속성으로는 alt, coords, href,shape,target 이 있다.
alt는 영역에 대한 대체 텍스트 이다.
coords는 영역의 좌표값을 설정하는 것이다. 이것은 그림판의 마우스로 한곳을 가져가면 그림판 왼아래에 좌표가 나오는데 그값으로 설정한다.
href는 영역 클릭시 구동되는 하이퍼 링크를 설정한다.
shape는 영역의 모양을 설정하는데 default , rect,circle,Poly가 있다.
default는 지정되지 않는 영역을 의미하고 rect는 사각형을 의미하며(x1,y1,x2,y2)로 사각형 대각선 꼭지점을 설정한다.
circle은 원이며 (x1,y1,r)원의 중점을 설정하고 그리고 반지름 길이를 넣어주면 된다.
poly는 여러가지도형을 만들수있는데 그만큼의 꼭지점좌표를 넣어주어야한다.(x1,y1,x2,x3,....)

<audio>는 소리컨텐츠를 정의하고 가능한 순서대로 한번만 실행한다.
속성은 autoplay,controls,loop,muted,preload,src가 있다.
autoplay는 오디오 파일이 준비되면 자동실행 여부이다.
controls는 오디오 조정화면 표시여부이다.
loop는 무한반보 실행 여부지정이다.
muted는 무음상태 지정이다
preload는 웹 페이지가 로드될때, 오디오 파일의 로딩상태 설정이다.
src는 오디오 소스파일의 URL이다.
<source>는 멀티미디어 다수컨텐츠의 소스를 정의한다.(한번만 실행한다.)
속성은 src,type,media가 있다.
src는 오디오URL지정이다.
type는 미디어 자원의 MIme-type이다.
media는 미디어 파일을 위한 미디어쿼리 지정이다.

<video>는 동영상 콘텐츠를 정의한다.
속성은 autoplay,controls,loop,muted,preload,src,poster,height,width가 있다.
autoplay는 비디오 파일이 준비되면 자동실행 여부지정이다.
controls는 비디오 조정화면 표시여부이다.
loop는 무한 반복 실행 여부 지정이다.
muted는 무음상태 지정이다.
preload는 웹페이지가 로드될 때, 비디오 파일의 로딩 상태 설정이다.
src는 비디오 소스파일의 URL이다.
poster는 재생준비중 표시할 포스터 이미지 파일경로 이다.
height는 비디오 재생 높이설정이다.
width는 비디오 재성 너비 설정이다.
